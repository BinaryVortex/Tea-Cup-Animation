# Tea Cup Animation

A small, lightweight CSS animation of a tea cup built with plain HTML and CSS. Perfect for learning CSS animations and practicing layout, transform, and keyframe techniques.

![Tea Cup Animation Screenshot](./Screenshot%202024-05-18%20004104.png)

## Demo
Open the `index.html` file in your browser to see the animation in action. No build tools or server required — just double-click `index.html` or serve the folder with a static file server.

## Features
- Pure HTML & CSS (no JavaScript)
- Smooth, looped tea cup animation using CSS keyframes
- Clean, commented styling for easy learning and modification

## Files
- `index.html` – the demo page
- `style.css` – all styles and animations
- `Screenshot 2024-05-18 004104.png` – preview image included in the README
- `Screenshot 2024-05-18 004136.png` – additional screenshot

## How to run
1. Clone the repository:

   git clone https://github.com/BinaryVortex/Tea-Cup-Animation.git

2. Open `index.html` in your browser.

Or use a simple HTTP server (recommended for consistent assets handling):

- Python 3.x:

  python -m http.server 8000

- Node (http-server):

  npx http-server .

Then open http://localhost:8000 in your browser.

## Customize
- Edit `style.css` to change colors, sizes, timing, or keyframe behavior.
- Duplicate the cup elements or adjust transforms to create different animation styles.
